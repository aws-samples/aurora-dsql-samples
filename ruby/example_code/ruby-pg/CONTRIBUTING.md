# AWS DSQL Node-Postgres code examples

## Overview

The code examples in this topic show you how to use the Ruby-pg work with AWS DSQL. 

## Run the examples

### Prerequisites

* Ruby version >=2.5 is needed

### Setup test running environment 

```sh
bundle install
```

### Run the example tests

```sh
rspec
```
